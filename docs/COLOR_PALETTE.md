# ClearGO Color Palette

## Primary

| Name       | Hex       | Usage                          |
|------------|-----------|--------------------------------|
| White      | `#FFFFFF` | Pure white, text on dark       |
| Platinum   | `#FAF8F5` | Page backgrounds (e.g. /epics) |
| Cast Iron  | `#37352A` | Nav bar, dark surfaces, text on brass |

## Signature Accents

High-impact accent colors for key highlights and brand moments.

| Name   | Hex       | Usage                    |
|--------|-----------|--------------------------|
| Alloy  | `#FFA680` | Conditional Go, highlights |
| Copper | `#FF7A52` | Logo, primary accent, CTAs |

## Secondary Colors

Complement the primary palette; add flexibility and depth. Use for contrast and hierarchy without overpowering the core identity.

| Name       | Hex       |
|------------|-----------|
| Bronze     | `#6C3A2A` |
| Verdigris  | `#9EB4AB` |
| Steel      | `#697771` |
| Pewter     | `#A1B4BA` |
| White Gold | `#F4EBD7` |
| Brass      | `#C3B497` |

## CSS Variables

Brand colors are defined in `src/app/globals.css` under the "Brand palette" section and can be referenced as:

- `var(--color-white)`
- `var(--color-platinum)`
- `var(--color-cast-iron)`
- `var(--color-alloy)` / `var(--color-conditional-alloy)`
- `var(--color-copper)`
- `var(--color-bronze)`
- `var(--color-verdigris)`
- `var(--color-steel)`
- `var(--color-pewter)`
- `var(--color-white-gold)`
- `var(--color-brass)` / `var(--color-accent)`
